# Task Log: Release 0.1.2

## Goal

Cut a clean release from current `main` after the pack-artifact fix so the published version matches the commit users actually install.

## Why

- `v0.1.1` already points at an older commit.
- Current `main` includes the fresh-runner pack fix that CI now depends on.
- The next publishable version should move forward cleanly instead of trying to reuse `0.1.1`.

## Scope

- Bump package, lockfile, and plugin manifest to `0.1.2`.
- Refresh pinned install commands and release docs to `0.1.2`.
- Re-run OpenClaw lab verification and package-artifact checks from the release candidate.
- Tag and publish `v0.1.2` from `main`.

## Expected Result

`clawseatbelt@0.1.2` becomes the first version whose tag, release docs, and current `main` all agree on the same publishable artifact.

## Verification

- `npm run verify:openclaw-lab:docs`
- `npm run release:check`

## Result

- Disposable OpenClaw install stayed warning-free with allowlist pinning intact.
- Pack verification produced `clawseatbelt-0.1.2.tgz` at 31982 bytes with 129903 bytes unpacked.
- Release metadata, pinned install commands, and package-proof docs now point to `0.1.2`.
